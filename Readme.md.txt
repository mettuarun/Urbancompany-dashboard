# Customer Vendor Dashboard

This project is a Power BI dashboard connected to a SQL Server database (`sm_db`) using the table `dbo.customer_vendor_final`.

## 📊 Tools Used:
- Power BI
- SQL Server (SSMS)
- DAX
- GitHub

## 📁 Files Included:
- `CustomerDashboard.pbix` - Main Power BI file
- `CustomerVendorInsights.sql` - SQL queries for data analysis
- `README.md` - Project overview and documentation

## 🧠 Insights Covered:
- Total Revenue by Service Category
- Order Count by Status
- Monthly Revenue Trends
- Vendor Performance
- Cancellation Reasons

## ▶️ How to Use:
1. Open `CustomerDashboard.pbix` in Power BI Desktop
2. Ensure your SQL Server (`sm_db`) is running
3. Refresh the data to load updated insights
